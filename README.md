# Ficha de Atendimento SAMU - Microrregião Nova Lima

Este projeto é um formulário digital para registro de Atendimento Pré-Hospitalar (APH) do SAMU, criado para facilitar a coleta de dados pelas equipes.

## Funcionalidades
* Preenchimento de dados do paciente, indicadores clínicos e procedimentos.
* Seleção dinâmica de equipe (USB/USA) com CBOs atrelados.
* Integração direta com Google Sheets para geração posterior de arquivos BPA-C para o SIA/SUS.
